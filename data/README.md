## Data

This project uses the public "Online Retail" transactional dataset (UCI Machine Learning Repository).

### Files
- `data/raw/` contains the original dataset (not committed to GitHub)
- `data/processed/online_retail_cleaned.csv` is generated by the cleaning notebook
- `data/processed/marketplace.db` is generated by the load-to-sqlite notebook
- `data/processed/exports/` contains KPI extracts used for Power BI

### How to reproduce
1. Download the dataset and place it at:
   `data/raw/Online Retail.xlsx`
2. Run notebooks in order:
   - `01_data_cleaning_and_features.ipynb`
   - `02_load_to_sqlite.ipynb`
   - `03_run_kpis_export_csv.ipynb`
